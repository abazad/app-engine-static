App Engine Static

Quickly create and host static files on app engine.

Tips:

  • Only modify templates/ & static/ folder to your needs and all pages will be generated from its structure.
  • templates/layout/ is ignored from generated pages, but you can still use any folder for creating templates and macros.
  • Run: python generate.py to update your static files.
  • All generated html goes to /gen/ it's not recommended to add this in your repository unless you use git to deploy.
